# Essbase Calc Script Linter and Syntax support

## Install
1. Download zip
2. Copy folder into %USERPROFILE%\.vscode\extensions 
3. Reload VSCode

## Supported error checking

1. FIX/ENDFIX balancing
2. IF/ELSE/ELSEIF/ENDIF balancing
3. Rogue comma detection

## Supported syntax

See [essbase language file](https://github.com/haoq-repo/vsc-essbase-linter/blob/main/syntaxes/essbase.tmLanguage.json) for full list of supported syntax items and their category.
